# Mohini Khatiyan — Astrologer & Psychic Website

A beautiful, fully responsive website for **Mohini Khatiyan** — Certified Numerologist, Tarot & Dice Reader, Runes Practitioner, Hanuman Healer & Psychic Consultant.

🔗 Instagram: [@healingssbymohini](https://www.instagram.com/healingssbymohini/)

---

## ✦ Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS** (styling)
- **shadcn/ui** (UI components)
- **Lucide React** (icons)
- **React Router DOM** (routing)

---

## ✦ Sections

| Section | Description |

## ✦ Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
